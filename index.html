Things I've Built

01
AI / Python My Best Work
🤖
AI Document Chatbot

A chatbot that reads a PDF and answers questions about it — like having a conversation with a document. I built the full pipeline myself: load the PDF, split it into chunks, create embeddings, store in a vector database, and retrieve the right context before answering.

What I learned building this
  • How RAG (Retrieval-Augmented Generation) actually works end-to-end
  • Using HuggingFace embeddings and ChromaDB for semantic search
  • Connecting to the Groq LLM API and managing conversation memory
  • Deploying an interactive Python app with Streamlit
