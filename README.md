## 🚀 Projects

### 🤖 AI Document Chatbot
A ChatGPT-style chatbot that reads PDF files and answers questions from them using Retrieval-Augmented Generation (RAG).

**What I built:**
- Full RAG pipeline — load PDF → chunk → embed → store → retrieve → answer
- Conversational memory for multi-turn dialogue
- Semantic search using HuggingFace embeddings + ChromaDB
- Deployed with Streamlit

**Stack:** Python · LangChain · Groq · HuggingFace · ChromaDB · Streamlit  
🔗 [github.com/Pavankumar-std/First_chatbot](https://github.com/Pavankumar-std/First_chatbot)
